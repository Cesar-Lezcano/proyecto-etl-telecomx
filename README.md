# Proyecto ETL - Análisis de Datos TelecomX

## Descripción del Proyecto

Este proyecto consiste en un proceso completo de ETL (Extract, Transform, Load) aplicado a datos de clientes de la empresa TelecomX. El objetivo es limpiar, transformar y analizar la información para obtener insights sobre el comportamiento de los clientes.

## Tecnologías Utilizadas

- Python (Pandas, Matplotlib, Requests)
- Google Colab
- Git / GitHub

## Proceso Realizado

### 1. Extracción
- Carga de datos desde archivo JSON en GitHub
- Normalización de estructura anidada

### 2. Transformación
- Limpieza de espacios en blanco
- Conversión de tipos de datos
- Creación de nuevas columnas
- Manejo de valores inconsistentes

### 3. Carga y Análisis
- Exportación a CSV y Excel
- Análisis descriptivo de variables
- Visualización de distribuciones
- Identificación de patrones

## Resultados Obtenidos

- Base de datos limpia y estructurada
- 7,267 registros procesados
- 20+ variables analizadas
- Visualizaciones de las principales métricas
